# Supplementary reproducibility data

This folder was added to the final CPC manuscript package.

Contents:
- data/: CSV files used to regenerate Figures 1-11.
- tables/: CSV files corresponding to manuscript tables.
- notebooks/reproduction_notebook_improved_cpc.ipynb: final Jupyter notebook.
- README_CSV_DATA.md: figure-data manifest and usage notes.

The manuscript figures are stored separately in the main figures/ directory of the LaTeX package.
